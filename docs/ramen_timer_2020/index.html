HOW TO USE

  1. 「RAMEN-NAME」に作りたいラーメンを入力してね。
    詳しい書き方とか使える単語は、下の「DICTIONARY」を見てね。
  2. 「SET」ボタンを押してタイマーをセットしよう。そのまんまだね。
  3. 心の準備ができたら「START」ボタンを押して、タイマーをスタートさせよう。そのまんまだね。
  4. タイマーを止めたいときは「PAUSE」を押そう。そのまんまだよ。
  5. タイマーを最初に戻したいときや、違うラーメンを食べたくなって入力しなおしたときは「SET」ボタンを押してね。つまり無限に使えるよ。

DICTIONARY

※カタカナにするのがメンドクサイ場合、ひらがなでも大丈夫です。

OMAKE

  • 硬さ継承
    ONにすると硬さが手前までの工程にも反映されます。
    例えば、「インスタントインスタントバリカタラーメン」を作るとき、一番最初の工程もバリカタで作ることができます。(通常時の「インスタントバリカタインスタントバリカタラーメン」と同じ)
    硬さが継承されている工程で硬さを指定すると、硬さを「掛け合わせる」ことができます。
    (硬すぎて1工程が1ピコ秒未満であったり、柔らかすぎて完成に1000年以上かかるラーメンは作れません。)
  • ラーメンの相対速度
    タイマーが、俗にいう「ウラシマ効果」により生じる時間のズレを考慮した速さになります。
    「自分(このツールを利用してるデバイス)」から見た「ラーメン」の速度を入力してください。
    ラーメンが加速している場合は正数、自分が加速している場合は負数となります。
    (光速 (1,079,252,848.8 km/h) を超えたラーメンは作れません。)
